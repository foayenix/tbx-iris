# Iris App Assets

This directory contains all assets used in the Iris app.

## Directory Structure

- **images/** - App images, logos, and illustrations
- **styles/** - Art style thumbnails for AI generation preview
- **animations/** - Lottie animations for loading and transitions

## Assets Needed

### images/
- `app_icon.png` - App icon (1024x1024)
- `onboarding1.png` - Onboarding illustration 1
- `onboarding2.png` - Onboarding illustration 2
- `onboarding3.png` - Onboarding illustration 3
- `onboarding4.png` - Onboarding illustration 4

### styles/
Art style preview thumbnails (512x512 recommended):
- `neon_cyber.jpg` - Neon Cyber style preview

### animations/
- `loading.json` - Loading animation (Lottie)
- `success.json` - Success animation (Lottie)
- `iris_scan.json` - Iris scanning animation (Lottie)

## Adding Assets

1. Place assets in the appropriate directory
2. Ensure assets are optimized for mobile (compressed)
3. Use appropriate formats (PNG for transparency, JPG for photos, Lottie JSON for animations)
4. Assets are already declared in pubspec.yaml
